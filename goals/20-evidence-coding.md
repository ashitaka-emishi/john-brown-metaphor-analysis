# Goal 20 — Build the Evidence Matrix

## Objective

Code the corpus with explicit categories and traceable evidence records.


## Operating contract

Read `AGENTS.md`, the `$scholarly-research` skill, and the files named below.
Work in checkpoints. Update `logs/progress.md`.
Do not proceed past a human gate.
Run `python -m lincoln_research validate` and `pytest` before completion.


## Required work

1. Create stable source and evidence identifiers.
2. Extract passages without losing context.
3. Populate `research/data/evidence-matrix.csv`.
4. Code nation ontology, war frame, death frame, temporal structure, religious register, political function, and bodily reality.
5. Record an alternative reading for every thesis-bearing row.
6. Mark disconfirming evidence.
7. Compare manuscript variants.
8. Compare Lincoln and Everett with the same codebook.
9. Produce `research/notes/coding-memo.md`.
10. Produce `research/reviews/coding-review.md`.

## Completion criteria

- Every key textual claim has source, locator, and passage.
- No interpretation is presented as quotation.
- Alternative readings are recorded.
- Contradictions remain visible.
- Validation and tests pass.
