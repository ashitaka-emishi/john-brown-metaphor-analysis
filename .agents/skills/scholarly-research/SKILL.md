# Scholarly Research Skill

## Workflow

1. Read `AGENTS.md`, the active goal, project configuration, and dossier.
2. State the checkpoint in `logs/progress.md`.
3. Decompose work into verifiable tasks and identify gates.
4. Acquire sources under `references/source-acquisition.md`.
5. Preserve raw data, provenance, hashes, rights notes, and transformations.
6. Normalize content separately.
7. Code evidence while separating quotation, observation, interpretation, and conclusion.
8. Map every material claim to evidence and counterevidence.
9. Draft only from approved claims.
10. Run source, historical, rhetorical, anachronism, and adversarial reviews.
11. Validate with repository commands.
12. Never declare success when required checks fail.

## Network work

Create source-specific adapters. Require dry-run, timeout, conservative delay, descriptive user agent, and explicit output path. Never evade robots.txt, rate limits, authentication, paywalls, CAPTCHAs, or terms.

## Invocation

Invoke with `$scholarly-research`, or allow Codex to select it when the task matches.
